## 繁體中文說明

**源和紅茶**是支援日文及繁體中文的自由字型，基於 [獅尾四季春加糖](https://github.com/max32002/swei-spring) 以及 [源暎ラテミン/源暎ラテゴ](https://okoneya.jp/font/genei-latin.html)。

獅尾四季春加糖 JP 版未有修改日語假名，導致假名與漢字混排時效果欠佳。本字型因此採用：

- **源暎ラテミン**：日語假名及英數字元
- **獅尾四季春加糖**：漢字  

組合成一款風格接近 **MinGo / Typos** 的字型。

### 字重

本字型僅提供兩個字重：

#### Regular

- 基於 Linux Biolinum 的原始字重（源暎ラテゴ的英數字為加粗後的 Linux Biolinum）。
- 日語假名則取自調幼後的源暎ラテゴ。
- 由於黑體與明體的粗細差異，漢字採用的是獅尾四季春加糖 **SemiBold**。

#### Medium

- 基於源暎ラテゴ的原始字重。
- 源暎ラテゴ的漢字來自思源黑體 Medium，故定為 Medium。
- 由於黑體與明體的粗細差異，漢字採用的是獅尾四季春加糖 **Bold**。

### JP版

JP版為日標字型：

- 假名與全形標點寬度與漢字相同。

PJP版亦為日標字型，但：

- 假名與全形標點為比例字寬。

兩者的英數字皆為比例字，不提供等寬版本。

### TC版

TC版以獅尾四季春加糖 JP 為基礎，修改成較接近傳統印刷體的字形。
純屬業餘修改，如有修改效果欠佳，請見諒。

#### 修改原則

- 簡化字原則上不作修改。
- 一般簡筆俗字及異體字，尤其無明顯楷化特徵者，不予修改。
- CJK 擴充區多數不修改，僅修改較常用的字。
- 為了節省修改時間，盡量沿用日標、台標或陸標以及相容漢字區旣有的字圖，所以部份部件使用了折衷的寫法（例如「甚」部件）。
- 由於獅尾四季春本身是批量處理而成，所以瑕疵頗多。我無意花上過多時間修改此字型，因此部份部件或字是明知而不作修改，詳見下方清單。
- 字源分離造成的多碼字依五大碼碼位是否符合字理去決定是否調整。

  - 該字五大碼有收錄，而非五大碼者較不合字理，非五大碼者不修改，例如：
    - 閲 U+95B2 → 不改為 閱 U+95B1

  - 該字五大碼有收錄，而五大碼者較不合字理，五大碼者會修改，例如：
    - 為 U+70BA → 更換成 爲 U+7232
    - 教 U+6559 → 更換成 敎 U+654E

  - 五大碼大致合乎字理，而 Big5-HKSCS 另收錄更合乎字理者，兩者皆保留，例如：
    - 舉 / 擧
    - 櫸 / 欅 （另有「𣟱 U+237F1」，右下方三橫）

    以上兩者維持分離。

    - 攑 / 㩮，前者 Big5-HKSCS 有收錄，後者為J源字。前者採用康熙字典收錄的三橫寫法，不合併為後者。

#### 以下部件不作修改

如果有不在下述清單上的字，而該字看似是改漏（例如日標新字體），或極常用異體字誤被省略，又或者是擴充區的常用漢字（例如名人人名、極常用粵台客語用字之類），可以開issue，並說明原因，但不保證會修改。

##### 因涉及字數過多

- 「上」部件（如辛、音）
- 「良」部件
- 「乎」部件
- 「曰」部件
- 「禸」部件
- 上方避讓的「尚」部件

##### 因統一碼合併問題

- 「蒦」字由於統一碼錯誤合併草頭和羊頭兩字所以不作改動。
- 「朊」（U+670A）字本該從月。由於在分拆出擴充A區從肉月的「䏓」（U+43D3）前，已於五大碼年代被大量文書取作肉月用，且從月者不屬現代用字，故兩者皆保留為肉月。

##### 因修改效果欠佳

- 「舛」部件的夂
- 「歰」部件
- 「禽」部件
- 包括且不限於綌、攧、釉、辤、閿、嚺、開字等

##### 因意義不大

- 旡部件
- 部份牙部件
- 左側而又不是⿺的麥 / 鼠部件

##### 其他原因

- 部份幵部件
- 并部件不常用的字
- 罙部件

### 對原字型的額外修改

在支援 OpenType **locl** 特性的程式（例如 Microsoft Word）中，當編輯語言設為繁體中文、簡體中文或日文時，基本標點符號會依語言自動切換樣式。

本字型 **現已支援** U+FE00 / U+FE01 SVS 標點符號選擇符。

在不支援 locl 的程式中，JP版會使用日文樣式標點，TC版會使用置中樣式標點。

此外，在 JP 及 PJP 版中，如果 **locl** 特性設定為中文，部份（但非全部）日本新字體會轉換為較接近中文習慣的寫法。

#### 對源暎ラテミン的修改

- 源暎ラテミン V2 為相容 GDI 環境的直排標點顯示，刪除了大量 GSUB 表。本字型補回部分功能（如 U+309A 組合鼻濁音的字圖等），經 Office 2003 測試直排顯示正常。
- 補上 Linux Biolinum 的 Kerning。（由於是 GPOS，所以 PowerPoint 等程式不支援）
- PUA 區的斜體感嘆號與問號可透過在 **！、？、‼、⁇、⁈、⁉** 後加上變體選擇符 **U+E0101** 存取。
  此做法不屬 Unicode 規範，但在部分情況下總比 PUA 及回收標誌較方便使用。
- PUA 區 以及 回收標誌區的感嘆號與問號於 P 版改為比例字寬。

#### 對獅尾四季春加糖 JP 的修改

- 移除筆劃重疊，以確保列印時正確柵格化以及新版 Microsoft Office 之類程式正確計算 Faux Bold。
  > 這些重疊與可變式字型毫無關聯，大部份是加粗筆劃時產生。這類重疊在靜態字型會影響相容，將其保留唯一價值僅是令編輯字型時會比較方便。
- 修正部份字符的直排顯示。
- 修正以下字圖誤植：

  - 「樂」「寧」「率」「郎」「燐」「憐」「鱗」誤植為台標字型
  - 「郞」（U+90DE）誤植為「郎」（U+90CE）
  - 「說」（U+8AAA）誤植為「説」（U+8AAC）
  - 「矏」誤植為「矈」
  - 「䞅」誤植為「賑」

- 「冤」改用 JIS2004 字形。
- 修正鳵、僷、膚、簴、噓、驉、魖、鰄、鱡、錻、嵂等字的嚴重變形。
- 修正部份去三角造成的輕微變形，以及為部份處理得特別差的常用字額外人手去三角（例如三個相鄰平行橫筆保留了一個三角）。
- 新增「㋿」符號，另加入㍻、㍼、㍽、㍾及㍿的直排字樣。
- 「门」「复」「关」預設字圖改為中國簡化字。原日本字圖可透過 **IVS U+E0100** 變體選擇符存取。
- 補全 CJK 部首。
- 補上 Unicode 後來新增的 BMP 急用漢字。
- 補充部份 SIP 漢字，詳見文末（部份字圖修改自[源樣明體](https://github.com/ButTaiwan/genyo-font)）。

### 注意事項

- 本字型僅 **有限度支援 IVS 漢字**，未覆蓋所有 IVS 漢字，亦有部份 IVS 字會直接使用思源宋體字圖（未去三角或加粗橫筆）。
- 由於獅尾四季春加糖使用機械批量去三角，所以會出現各種瑕疵，恕未能保證成品品質。
- 由於本字型使用機械批量移除筆劃重疊，所以可能會出現各種破字。此類情況請及時回報。
- 除非字形因去三角錯誤導致筆劃出現**明顯畸形**，否則不接受與「去三角」相關的修改要求。

## 日本語（概要）

**源和コウチャ**は、日本語と繁体字中国語に対応したフリーフォントです。[獅尾四季春加糖](https://github.com/max32002/swei-spring) と [源暎ラテミン・源暎ラテゴ](https://okoneya.jp/font/genei-latin.html) をベースに制作されています。

獅尾四季春加糖JPは仮名が調整されていないため、仮名と漢字を混植した際に違和感が生じる場合があります。
本フォントでは、

- **源暎ラテミン**：英数字・仮名
- **獅尾四季春加糖**：漢字

を組み合わせ、ミンゴ・タイポス系書体に近い雰囲気のフォントとして構成しています。

### ウェイト

本フォントは以下の2ウェイトのみを提供しています。

- **Regular**
- **Medium**

実際にはゴシック体と明朝体の太さの違いを考慮し、獅尾四季春加糖側では次のウェイトを使用しています。

- Regular → SemiBold
- Medium → Bold

### 繁体字フォント（TC）

TCはJPをベースに、康熙字典体に近い字形へ調整したバージョンです。
簡体字や拡張漢字の多くは変更していません。

### 日本語フォント（JP・PJP）

本フォントには完全な等幅版はありません。
欧文と数字はプロポーショナルですが、次の2種類のバージョンがあります。

- **JP**：仮名と全角記号は等幅
- **PJP**：仮名と全角記号もプロポーショナル

OpenType **locl** に対応するアプリケーションでは、編集言語に応じて句読点と一部の漢字のグリフが切り替わります。

### 主な変更内容

#### 源暎ラテミン

- 源暎ラテミン V2 では、GDI 環境での縦書き句読点表示対応のため GSUB 情報が大幅に削除されています。
  本フォントでは一部（U+309A で結合した鼻濁音グリフなど）を補っています。Office 2003 で縦書き表示は確認済みです。
- Linux Biolinum のカーニングを追加。（GPOS であるため、PowerPoint などではサポートされていません。）
- PUA 領域の斜体の感嘆符・疑問符は、**！、？、‼、⁇、⁈、⁉** の後に変体選択子 **U+E0101** を付加することでアクセスできます。
- リサイクルマーク領域および PUA 領域の感嘆符・疑問符は、P 版ではプロポーショナルになります。

#### 獅尾四季春加糖JP

- 一部のラスタライザーに対応するため、ストロークのオーバーラップを削除。
- 縦書き表示の修正。
- 一部誤字形や変形文字の修正。
- CJK 部首や一部 Unicode 追加漢字を補完。
- 「门」、「复」、「关」のグリフが簡体字中国語に変更されました。元の日本語グリフは、**IVS U+E0100** 変体選択子でアクセスできます。

### 注意

- 獅尾四季春加糖の漢字は機械的な処理でウロコを削除しているため、字潰れなどがある場合があります。品質は保証しません。
- オーバーラップの削除は機械的な処理で行っているため、字形が変形している場合があります。問題を見つけた場合はご報告ください。
- IVS漢字は **部分的にのみ対応**しています。
- 一部のIVS漢字は源ノ明朝のグリフをそのまま使用しています。
